Hello World

To get started with writing Java, open a text editor like Sublime Text or Visual Studio Code and write your first "Hello World" Java Code.

First we need to create a file called HelloWorld.java.
In the file, we need to write the following:
System.out.println("Hello World");
Then we have to compile it with javac. Install JDK!
java HelloWorld.java
Don't forget to compile it and then run the code with java Hello_World.java!!

OOP

OOP or Object Orientated Programming is a key idea in Java. We are not going to go into detail here just the key points. They are the following:

  • Abstraction - In Java, the concept of abstraction means simple things such as objects and classes.
  • Encapsulation - In Java, this is about keeping fields within classes private, then providing access to them via public methods. This is a protective barrier that keeps the data and code safe within the class itself.
  • Inheritance - This lets programmers create new classes that share some of the attributes of existing classes. This lets us build on existing works without needing to re-create code each time we need to re-use code.
  • polymorphism - This Java OOP concept lets programmers use the same word to mean different things given different contexts. One common form of polymorphism in Java is called method overloading. That's when the different meanings are dependent on the values of the supplied variables.

Classes And Objects

Here is example with sample code implementing the Bicycle class, to give an overview of how to declare a class.

  • Classes
    public class Bicycle
    //the Bicycle class has three fields
    public int cadence;
     public int gear;
     public int speed;

     // The Bicycle class has one constructor
     public Bicycle(int startCadence, int startSpeed, int startGear) {
     gear = startGear;
    cadence = startCadence;
    speed=startSpeed;
     }

    the Bicycle class has four methods which we are listing out below:

    Method #1:
    public void setCadence(int newValue) {
    cadence = newValue;
    }
    Method #2:
    public void setGear(int newValue) {
    gear = newValue;
    }
    Method #3:
    public void applyBrake(int decrement) {
    speed -= decrement;
    }
    Method #4:
    public void speedUp(int increment) {
    speed += increment;
    }

Annotations

Annotations, a form of metadata, provide data about a program that is not part of the program itself. Annotations have no direct effect on the operation of the code they annotate.
